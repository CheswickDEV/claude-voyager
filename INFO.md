# Claude Voyager

## English

Claude Voyager is a Firefox extension that adds productivity tools directly into the claude.ai interface. It includes timeline navigation, chat folders, a prompt library, export options, adjustable chat width, tab title sync, and formula copy — all designed to blend seamlessly into Claude's native UI.

## Deutsch

Claude Voyager ist eine Firefox-Erweiterung, die Produktivitätswerkzeuge direkt in die claude.ai-Oberfläche integriert. Enthalten sind Timeline-Navigation, Chat-Ordner, eine Prompt-Bibliothek, Export-Optionen, anpassbare Chat-Breite, Tab-Titel-Sync und Formel-Kopieren — alles nahtlos in Claudes natives Design eingebettet.
